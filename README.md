# Snake
A classic mini game
# Environments 
C++ 11
# Build
## Windows
install MinGW from [here](https://sourceforge.net/projects/mingw/)
then configure the environment variable
run commandline
`g++ main.cpp -o <output filename>`
In cmd
## Linux
Run the commandline to install g++
`sudo apt-get install g++`
then type this commandline
`g++ main.cpp -o <output filename>`
